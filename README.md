# Olympic Games Analysis Dashboard

A Power BI dashboard analyzing historical Olympic Games data from the 1880s to the 2020s.

## Overview
Interactive multi-page report covering medal trends, country performance, athlete demographics, and sport-level breakdowns across 150+ nations and 140+ years of Olympic history.

## Features
- **KPI Cards** – Total medals (20,281), Gold (6,631), Silver (6,598), Bronze (7,052)
- **Scatter Plots** – Games participated vs. medals won (Summer & Winter separately)
- **Choropleth Map** – Country-level medal distribution via Bing Maps
- **Trend Chart** – Gold, silver & bronze counts from 1880–2020
- **Stacked Bar Charts** – Medal breakdown by country, sport, sex, and season
- **Pie Chart** – Medal share by athlete sex (71.71% male / 28.29% female)

## Tools & Techniques
- **Tool:** Microsoft Power BI
- **Data Modeling:** DAX measures for aggregated medal calculations
- **Visuals:** 7+ chart types across multiple report pages with drill-through
- **Design:** Dark navy theme for visual consistency
